# Print Easy System

## Overview

Print Easy is an online printing management system designed to make the printing process easier and faster for students and other users. The project was created to solve common problems faced during printing, such as long queues, document mix-ups, delays, and difficulty in managing assignments and notes.

With Print Easy, users can upload documents online, select print options, and place print requests without standing in line. The system helps save time, improve organization, and provide a smoother printing experience.

---

## Features

* User-friendly interface
* Online document upload
* Print request management
* Selection of print options (pages, copies, color, etc.)
* Reduced waiting time
* Organized document handling
* Easy access for students and staff

## Problem Statement

Students and office users often face issues while getting printouts, such as:

* Long queues at print shops
* Wastage of time
* Mixing of documents
* Difficulty managing multiple print requests

Print Easy solves these issues by providing an online platform where users can upload files and manage print requests efficiently.

---

## Objectives

* To simplify the printing process
* To reduce manual effort and waiting time
* To provide an organized printing system
* To improve user convenience

---

## System Workflow

1. User logs into the system
2. Uploads the required document
3. Selects print settings
4. Sends the print request
5. Admin/printer receives the request
6. Document gets printed and delivered
